All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Initial project structure and architecture
- Claude Code agent role with 9 specialized agent personas
- Qwen Code agent role with YOLO mode configuration
- VibeKanban task management role with Shortcut integration
- Gemini CLI role (placeholder for future implementation)
- Comprehensive documentation (README, CONTRIBUTING, role-specific READMs)
- Ansible playbook with modular role-based design
- Support for local and remote deployment
- Idempotent install/uninstall operations
- Workflow guides (Conventional Commits, Git Hooks, Shortcut ↔ VibeKanban)
- Apache 2.0 license

### Specialized Agents
- architecture-guardian: Clean Architecture enforcer
- backend-developer: Java 24+, Spring Boot 4.x expert
- data-engineer: ETL/ELT, big data specialist
- frontend-developer: React, Next.js, Flutter expert
- identity-security-developer: OAuth2, OIDC, passkeys expert
- infrastructure-engineer: AWS, GCP, Kubernetes specialist
- mobile-engineer: iOS, Android, cross-platform expert
- principal-engineer: Strategic arbiter and decision-maker
- secops-engineer: OWASP, security tooling expert
